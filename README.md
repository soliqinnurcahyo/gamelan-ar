# 🎵 Gamelan AR

Media Pembelajaran Alat Musik Gamelan Berbasis **Web Augmented Reality (WebAR)** untuk Siswa Kelas IV Sekolah Dasar.

## 📖 Tentang Project

Gamelan AR merupakan aplikasi pembelajaran interaktif berbasis Web Augmented Reality (WebAR) yang dirancang untuk membantu siswa mengenal alat musik gamelan secara lebih menarik.

Pengguna cukup membuka website melalui browser, kemudian mengarahkan kamera ke marker pada majalah pembelajaran. Setelah marker terdeteksi, aplikasi akan menampilkan model 3D alat musik gamelan beserta informasi dan audio penjelasannya.

Project ini dikembangkan sebagai tugas skripsi.

## ✨ Fitur

- 🎯 Marker Based Tracking menggunakan MindAR
- 🎨 Model 3D Alat Musik Gamelan
- 🔊 Audio Penjelasan Setiap Alat Musik
- 📖 Informasi Lengkap Alat Musik
- 📱 Berjalan langsung di Browser (WebAR)
- 👆 Rotasi Model 3D dengan Gesture
- ⚡ Loading Progress
- 📷 Scan Marker Interaktif

## 🚀 Instalasi

Clone repository

```bash
git clone https://github.com/soliqinnurcahyo/gamelan-ar.git
```

Masuk ke folder project

```bash
cd gamelan-ar
```

Install dependency

```bash
npm install
```

Menjalankan project

```bash
npm run dev
```

Build project

```bash
npm run build
```

Preview hasil build

```bash
npm run preview
```
